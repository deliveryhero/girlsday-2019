HTML & CSS Anleitung

HTML

<p> – Pagagraph

Mit <p> beschreibst du einen Textabsatz.

Beispiel:
<p>Hallo, ich bin ein Text</p>

<h1> – Überschrift 1

Mit <h1> beschreibst du die Hauptüberschrift.

Beispiel:
<h1>Überschrift</h1>

<h2>, <h3>, <h4>, <h5>, <h6> – Weitere Überschrift

Mit <h2> bis <h6> beschreibst du weitere Überschriften. Damit gibtst du deiner Seite struktur.

Beispiel:
<h2>Andere Überschrift</h2>
<h3>Noch eine Überschrift</h3>

<br> – Zeilenumbruch

Mit <br> kannst du einen Zeilenumbruch machen. Achte darauf, dass es hier keinen Schließenden "Tag" gibt.

Beispiel:
<p>Das ist ein Text <br> und hier bricht er um </p>

<strong> – Wichtiger Text

Mit <strong> beschreibst du einen teil von deinem Text, der wichtig ist. Standardmäßig wird er fett dargestellt.

Beispiel:
<p>Das ist ein <strong>wichtiger<strong> Text </p>

<em> – Hervorgehobener Text

Mit <em> beschreibst du einen teil von deinem Text, der hervorgeheben wurde. Standardmäßig wird er kursiv dargestellt.

Beispiel:
<p>Das ist ein <em>hervorgehobener<em> Text </p>

<ul> & <li> – Auflistung

Mit <ul> und <li> beschreibst du eine Liste. Das <li> Element beschreibt die einzelnen Listenelemente und <ul> die Liste selber

Beispiel:
<ul>
    <li>Das ist</li>
    <li>eine</li>
    <li>Auflistung</li>
</ul>

<div> & <span> – Neutrale Elemente

<div> und <span> sind beides neutrale Elemente. Das heißt, sie haben keine Bedeutung. Das ist perfekt um sie mit CSS zu benutzen. Der einzige Unterschied ist, dass das <div> Element ein Block ist (z.B. wie Überschrift und Absatz). Das <span> Element ist hingegen ein "inline" Element, also es kann innerhalb vom Text benutzt werden und bricht auch mit um.

Beispiel:
<div>Text</div>
<span>Text</span>
<div>Text mit <span>noch mehr Text</span></div>

CSS

Klasse

Eine Klasse beschreibt eine Ansammlung von "Styles". Sie beginnt in CSS mit einem Punkt. Dahinter muss man das Zeichen "{" benutzen. Damit sagt man, dass die Styles für die Klasse beginnen. Hinter allen Styles muss man dann "}" schreiben.

Beispiel: 

.meine-klasse {
  color: yellow;
  background-color: blue;
}

Styles

Styles beschreibt ihr mit so

name: wert;
Vor dem Doppelpunkt kommt der Name der Eigenschaft (unten ist eine Auflistung von möglichen Eigenschaften). Dahinter der Wert für diese Eigenschaft. Bei Farbe ist Wert die Farbe, bei Abständen ist es der Abstand mit Einheit (zum Beispiel Pixel [px]).
Achte immer auf das Semikolon (;) hinter jedem Style.

Farbe

Hintergrundfarbe

background-color: blue;
background-color: #ff0000; <-- rot

Vordergrundfarbe

color: blue;
color: #00ff00; <-- grün

Schrift

Schriftgröße

font-size: 16px <-- Schriftgröße von 16 Pixeln
font-size: 1em <-- Gleiche Schriftgröße wie das Element drumherum (relativ)
font-size: 2em <-- Doppelte Schriftgröße wie das Element drumherum (relativ)

Schriftgröße

font-family: "Arial" <-- Schriftart Arial

Abstände

Abstände aussen

margin: 30px; <-- auf allen Seiten
margin: 30px 15px; <-- oben und unten 30px und links und rechts 15px
margin-top: 30px; <-- Abstand oben
margin-right: 30px; <-- Abstand rechts
margin-bottom: 30px; <-- Abstand unten
margin-left: 30px; <-- Abstand links

Abstände innen

padding: 30px; <-- auf allen Seiten
padding: 30px 15px; <-- oben und unten 30px und links und rechts 15px
padding-top: 30px; <-- Abstand oben
padding-right: 30px; <-- Abstand rechts
padding-bottom: 30px; <-- Abstand unten
padding-left: 30px; <-- Abstand links

Größen

width: 300px; <-- Breite von 300 Pixeln
height: 200px; <-- Höhe von 200 Pixeln

Rahmen

border: 2px solid blue; <-- 2 Pixel Rahmen mit Farbe Blau
